# [Tripping](ENTER_URL_HERE)

## Tech stack

- Next.js 14
- Shadcn/ui
- Tailwind CSS
- TypeScript
- Zustand

## Starting the project locally

1. Clone the repository

   ```bash
   git clone https://github.com/tituswe/tripping.git
   ```

2. Install dependencies

   ```bash
   cd tripping
   npm install
   ```

3. Run the development server

   ```bash
   npm run dev
   ```
